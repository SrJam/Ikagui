# Geo-Recon
An OSINT CLI tool desgined to fast track IP Reputation and Geo-locaton look up for Security Analysts.

# Setup
This tool is compactible with:
* Any Linux Operating System (Debian, Ubuntu, CentOS)
* Termux

# Linux Install Requeriments

pip install -r requirements.txt  #You also can use the pip3

# Sample Simple Syntax Linux

root@kali:~/geo-recon# python geo-recon.py 123.456.789.10
or
root@kali:~/geo-recon# python geo-recon.py 123.456.789.10 -n


# Commands

python geo-recon.py --help or -h                         (Display this)

python geo-recon.py 123.456.789.10 --nmap or -n          (Nmap standard use)

python geo-recon.py 123.456.789.10                       (Standard use, infos about IP)

python geo-recon.py --commands or -c                     (Display commands availables)
